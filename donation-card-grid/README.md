# Nonprofit Donation Card Grid

A reusable three-card layout for nonprofit donation pages, sponsorship sections, planned giving pages, and community support campaigns.

## What it solves

This snippet creates a clean, responsive card grid that helps nonprofit websites present multiple giving options in a clear and organized way.

It was designed to solve common layout limitations in Squarespace built-in blocks, especially uneven card spacing, inconsistent content alignment, and button placement when each card has a different amount of copy.

## Best used for

* Nonprofit donation pages
* Sponsorship sections
* Planned giving pages
* Matching gift information
* Community support campaigns
* Squarespace code blocks
* WordPress custom HTML blocks
* Static landing pages

## Key features

* Responsive three-card layout
* Consistent spacing between cards
* Better copy alignment inside each card
* Buttons aligned cleanly across cards
* Mobile-friendly stacking
* Easy to customize with brand colors, icons, and links

## Notes

This is a generalized, reusable example based on common nonprofit website patterns. Client-specific content, branding, images, URLs, and private project details have been removed.

Replace the placeholder images, button links, colors, and copy with your own project content.
