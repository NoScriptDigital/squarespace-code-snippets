# Squarespace Text and Icon Marquee

A reusable HTML and CSS marquee that combines short text phrases with decorative icons in a continuous horizontal loop.

## What it solves

Squarespace includes native scrolling text features, but the built-in options may not provide enough control over repeated phrases, decorative separators, spacing, typography, animation timing, and responsive behavior.

This snippet creates a fully controlled text marquee with icons such as:

* ×
* ■
* ◆
* *
* Custom symbol characters

## Files

* `index.html`
* `styles.css`

## Notes

This is a generalized reusable example.

Client names, phrases, brand styling, colors, and private project details have been removed or replaced.
